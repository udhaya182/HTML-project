About the course

  • Offered in semester 2 only.
  • This blended course is aimed at enriching students with fundamental concepts in programming.
  • Students will learn to perform binary conversions, understand variables and data types, and create programs in C++ by using sequential, selection (if-else) and iteration (loops) structures.
  • Students will learn how to use mathematical functions in C++, the order of precedence for arithmetic and logical operators, increment and decrement operators, and cast operators.
  • The course will conclude with some exposure to creating a web page with HTML and CSS coding.
  • This is also a compulsory course for students intending to pursue computing science at degree level.
